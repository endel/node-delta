node-delta
===

An efficient delta compression algorithm for Node.js. (C++ addon)

Benchmarks
---

```
node-delta x 228,969 ops/sec ±3.87% (65 runs sampled)
fossil-delta x 178,216 ops/sec ±3.80% (76 runs sampled)
```
